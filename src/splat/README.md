# SPLAT modules

## Carry Cascade

Standalone, browser-native generative cover studio for the Symonic SPLAT reporting system.

- **Entry point:** `src/splat/carry-cascade/index.html`
- **Runtime:** p5.js 1.7.0 loaded from CDN
- **Output:** deterministic 4:5 PNG cover editions keyed by seed
- **Controls:** seed navigation, palette editions, cascade physics, cover-furniture toggles, and PNG export

### Operating sentence

> The splat is a propagation event: one clean origin, debasing downstream, cut at the limit.

### Palette contract

| Edition | Ground | Splat | Type / Cut |
| --- | --- | --- | --- |
| Cover | `#1e1c1a` | `#6e4aa6` | `#e9dfa3` |
| Amber | `#111009` | `#c97e08` | `#f2e4c4` |
| Receipt | `#1a1917` | `#9a9890` | `#e7e4da` |

The visual grammar deliberately maps body, spread, downstream fingers, satellites, and a draughtsman's cut line to SPLAT's proof-oriented cover system.
